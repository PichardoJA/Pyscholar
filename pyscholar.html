pyscholar.save_graph_pickle(G, path='', name_graph='')¶

Save graph in Python pickle format.

This function saves saves a graphic object in pickle format in a directory.

Parameters:path (String.) – It is the directory where the object pickle is located.
Returns:Return a graph G.
Return type:Return a NetworkX graph.
Example: 
>>> import pyscholar
>>> my_graph = psycholar.load_graph_pickle("~/dir/graph.pickle")
>>> my_graph
pyscholar.search_affiliation_by_id(list_scopus_id_affiliation)¶

pyscholar.search_author(list_scopus_id_author)¶
