pyscholar module¶

This is the main module of PyScholar.

pyscholar.check_years(min_year='', max_year='')¶

Return the filter if the given interval is correct otherwise returns None

pyscholar.count_citations_by_id_paper(list_scopus_id_paper)¶

Returns a dictionary where the key is the ID of the paper and the value associated with the key is the number citations

pyscholar.find_author_scopus_id_by_name(firstName='', lastName='')¶

Searches for an author scopus id given its name.

pyscholar.get_cache_papers()¶

Returns the scopus_authors_by_idpapers_cache dictionary which is a cache of papers consulted where the key is the id of the paper and the associated value is a set of authors who wrote the article.

pyscholar.get_cache_papers_by_authorid()¶

Returns the global variable scopus_papers_by_authorid_cache which is a dictionary of dictionaries where the first dictionary key is the id of the author and the key to the second dictionary is the time interval whose associated value is a set of papers.

pyscholar.get_citation_graph(list_scopus_id_paper, distance, directory='', name='')¶

Returns a tuple where the first element is the graph induced by papers and the second element is a list of sets where each set is a set of papers to distance d. and the last element is a set of papers not found.

pyscholar.get_coauthors(id_author, min_year='', max_year='', dict_knowledge={})¶

Returns a tuple with the nex elements, 1.-Id_author 2.-Set of co-authors associated with an id of an author. 3.-A dictionary where the key is the ID of the co-authors and the value associated is a set with the ids of the papers between the author and co-author.

pyscholar.get_coauthors_graph(list_scopus_id_author, distance, min_year='', max_year='', directory='', name='')¶

Returns a tuple where the first element is the graph induced by several authors and the second element is a list of sets where each set is a set of authors to distance d.

pyscholar.get_common_papers(id_author_1='', id_author_2='')¶

Returns the intercession of papers between two authors

pyscholar.get_ids_authors_by_id_paper(list_scopus_id_paper)¶

Returns a dictionary where the key is the ID of the paper and the value associated with the key is a list of the ids of the authors who wrote the paper

pyscholar.get_papers(list_scopus_id_author, min_year='', max_year='')¶

Returns a dictionary where the key is the ID of the author and the value associated with the key is a set of the ids of the papers that belong to the author in certain time interval.

pyscholar.get_references_by_paper(list_scopus_id_paper)¶

Returns a dictionary where the key is the ID of the paper and the value associated with the key is a set of the ids of the papers cited by the main paper

pyscholar.get_title_abstract_by_idpaper(id_paper='')¶

Returns a tuple with the id_paper,title and abstract of each paper

pyscholar.load_authors_from_file(directory='')¶

Returns a list of the authors who were in a file. Each line must have a valid scopus id.

pyscholar.load_papers_from_file(directory='')¶

Returns a list of the papers who were in a file.

pyscholar.search_author(list_scopus_id_author)¶

Returns a dictionary where the key is the ID of the author and the value associated with the key is a dictionary with the following keys: name, surname, h-index and coauthor-count.
